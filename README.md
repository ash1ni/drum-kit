# drum-kit
A drum-kit project built with the help of HTML, CSS and JavaScript. <br>
Instructions for the keys: <br>
- w - crash sound <br>
- a - tom1 sound <br>
- s - kick sound <br>
- d - snare sound <br>
- j - tom3 sound <br>
- k - tom4 sound <br>
- l - tom2 sound <br>

you can chekout the project here: https://ash1ni.github.io/drum-kit/
